Guides

This following collection of guides is designed to take you through various aspects of Baleen, including use, configuration and development.

Administrator's Guides

  • Configuring Logging
    This guide will take you through configuring the logging of Baleen.
  • Ordering of Annotators
    This guide discusses how to best order annotators in your pipeline.
  • Running Baleen with Additional Annotators
    This guide will take you through how to run Baleen with additional third party annotators.

Developer's Guides

  • Adding a new database
    This guide considers how to connect to a new databases type.
  • Developing Annotators with Resources
    This guide will take you through the development of a Baleen annotator with a shared resource.
  • Developing RegEx Annotators
    This guide will take you through the development of a Baleen annotator that uses Regular Expressions to extract entities.
  • Developing SharedResources
    This guide will take you through the development of a new shared resource, such as a new databases.
  • Using configuration
    This guide discusses how allow users to configure Baleen elements.
